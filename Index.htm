Exemplo de uso de tags HTML

Este é um exemplo de uso das tags HTML para criar uma página web.

Podemos utilizar a tag mark para destacar um texto importante.

Também podemos utilizar a tag small para diminuir o tamanho do texto.

A tag i pode ser utilizada para colocar o texto em itálico.

A tag u pode ser utilizada para sublinhar o texto.

A tag HTML pode ser utilizada para definir uma abreviação e fornecer uma descrição completa do termo.

A tag del pode ser utilizada para indicar texto que foi removido.

A tag strong pode ser utilizada para colocar o texto em negrito.

A tag sub pode ser utilizada para colocar o texto em subscrito.

A tag sup pode ser utilizada para colocar o texto em sobrescrito.
